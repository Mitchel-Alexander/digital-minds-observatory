# Digital Minds Observatory

A living map of research into whether artificial systems (AI, robots, brain
organoids) could ever be conscious, and what we would owe them if they were —
scholarly works published since 2010, tracked as the field grows.

**Live page:** https://mitchel-alexander.github.io/digital-minds-observatory/

The page is a single self-contained HTML file regenerated from the underlying
curated corpus (maintained separately, with a PRISMA-informed methodology).
Citation counts are verified per work; snapshot date is shown on the page.
